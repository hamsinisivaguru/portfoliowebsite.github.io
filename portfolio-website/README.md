## üìù Customization

### Update Your Information

1. **Name & Contact**: Edit the personal details in `index.html`
   - Line 35: Change "Kamsini Sivakuru" to your name
   - Line 67: Update email address
   - Line 68: Update GitHub username URL

2. **Add Your GitHub Profile**:
   - Search for `YOUR_GITHUB_USERNAME` in `index.html`
   - Replace all instances with your actual GitHub username
   - The link will open in a new tab automatically

3. **Update Your CV**:
   - Replace `cv.pdf` with your actual CV file
   - Make sure the filename is exactly `cv.pdf` in the same directory as `index.html`

4. **Modify Projects**:
   - Update project titles, descriptions, and links in the Projects section
   - Change the project URLs to your actual project links

5. **Customize Skills**:
   - Edit the skills list in the Skills section
   - Each skill card has an icon, title, and description
